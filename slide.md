#### **2. Gemini CLI 핵심 개념 및 기본 명령어 (15분)**

*   **(5:00-10:00) Gemini CLI 핵심 개념 이해**
    *   Gemini CLI란? (LLM이 결합된 차세대 명령 줄 인터페이스)
    *   기존 CLI와의 차이점: 단순 명령어 실행을 넘어 '대화'와 '추론'이 가능
    *   Gemini CLI의 강력함: 컨텍스트 인식, 파일 시스템 연동, 확장성
*   **(10:00-20:00) 실시간 데모: 기본 명령어 및 활용**
    *   **[데모 1] 간단한 질의응답:**
        *   `gemini "React에서 state와 props의 차이점은?"` (간단한 기술 질문)
        *   `gemini "정규표현식으로 이메일 주소 찾아줘"` (코드 생성 요청)
    *   **[데모 2] 파일 연동 (`@` 활용):**
        *   `@semina.md` 파일 내용 요약 요청 (`gemini "@semina.md 이 파일의 핵심 내용을 3줄로 요약해줘"`)
        *   코드 파일 분석 및 리팩토링 제안 (`gemini "@main.js 이 코드에서 개선할 점을 찾아줘"`)
    *   파일 연동의 중요성 강조: 내 프로젝트에 대한 깊은 이해를 바탕으로 답변

#### **3. Nano Banana 확장 기능: 이미지 생성 마법 (15분)**

*   **(20:00-25:00) Nano Banana 소개**
    *   Gemini CLI의 확장 기능 개념 설명
    *   Nano Banana: 터미널에서 이미지를 생성하고 편집하는 강력한 도구
    *   개발자에게 이미지 생성이 왜 필요한가? (발표 자료, 블로그, 앱 아이콘, 목업 데이터 등)
*   **(25:00-35:00) 실시간 데모: Nano Banana 활용**
    *   **[데모 3] 기본 이미지 생성:**
        *   `nanobanana generate "a blue bird logo for a tech company"`
    *   **[데모 4] 스타일 및 옵션 추가:**
        *   `nanobanana generate "a fox sleeping under a tree --styles=watercolor,sketch --count=2"` (다양한 스타일로 여러 장 생성)
    *   **[데모 5] 세미나 홍보 이미지 제작:**
        *   `@semina.md` 파일 내용을 참고하여 홍보 이미지 생성 요청
        *   `gemini "@semina.md 내용을 기반으로 'Gemini CLI' 세미나 홍보용 이미지를 만들어줘. '개발 생산성 혁명'이라는 문구가 포함되면 좋겠어."`
