## **Gemini CLI 핵심 개념**
- **LLM 기반:** 자연어 '대화'와 '추론'이 가능한 차세대 CLI
- **컨텍스트 인식:** 채팅 기록, 파일 내용을 기억하여 더 똑똑한 답변 제공
- **파일 시스템 연동:** `@` 기호를 통해 프로젝트 파일과 직접 소통
- **강력한 확장성:** `nanobanana`와 같은 확장 기능으로 능력 무한 확장

## **[종합 DEMO] 실제 개발 워크플로우**
- **상황:** 새로운 버튼 컴포넌트를 만들고, 테스트 코드를 작성한 뒤, 작업 내용을 문서화하는 과정
1. **프로젝트 구조 파악:** `ls -R`
2. **테스트 코드 자동 생성:** `gemini "@src/Button.js 와 유사한 테스트 코드를 만들어줘"`
3. **UI 아이콘 제작:** `nanobanana generate_icon "a simple checkmark icon"`
4. **작업 로그 자동 문서화:** `gemini "방금 진행한 작업을 마크다운으로 정리해줘"`
